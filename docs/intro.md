# 臭喜喜的笔记本

这是一个基于 Docusaurus 的个人博客 + 文档站。

## 你能在这里看到

- 现代化界面
- 文档与博客混合内容
- 标签系统
- 搜索能力
- 可直接部署到 GitHub Pages

## 站点结构

- **文档**：放系列文章、项目总结、长文笔记
- **博客**：放短篇文章、更新、碎碎念
